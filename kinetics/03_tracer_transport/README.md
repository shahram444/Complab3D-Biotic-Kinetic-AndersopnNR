# Scenario 03: Tracer Transport (Flow + Advection-Diffusion, No Reactions)

## Overview

This scenario combines **Navier-Stokes flow** with **advection-diffusion transport** of a single passive tracer through a parallel plate channel. No reactions occur. It validates that the coupled flow-transport solver correctly advects and disperses a solute front, and that the system reaches a uniform steady-state concentration when the inlet continuously supplies tracer.

## Reactions

None. The tracer is conservative (non-reactive).

## Input Parameters

| Parameter | Value | Units | Description |
|-----------|-------|-------|-------------|
| `delta_P` | 2.0e-3 | (lattice) | Pressure drop driving flow |
| `tau` | 0.8 | (lattice) | LBM relaxation time |
| `dx` | 1 | um | Voxel size |
| `D` (Tracer) | 1.0e-9 | m^2/s | Molecular diffusion coefficient |
| `biotic_mode` | false | -- | No microbes |
| `enable_kinetics` | false | -- | No biotic kinetics |
| `enable_abiotic_kinetics` | false | -- | No abiotic kinetics |
| `number_of_substrates` | 1 | -- | One passive tracer |

## Domain

| Property | Value |
|----------|-------|
| Dimensions (nx x ny x nz) | 40 x 15 x 15 |
| Geometry type | Parallel plate channel |
| Description | Solid bounce-back walls on top and bottom, open sides, flow driven in the x-direction by the pressure gradient |
| Material numbers | 0 = solid, 1 = bounce-back walls, 2 = pore space |

## Boundary Conditions

| Boundary | Species | Type | Value |
|----------|---------|------|-------|
| Inlet (x = 0) | Flow | Pressure (higher) | Set by `delta_P` |
| Outlet (x = nx) | Flow | Pressure (lower) | Reference pressure |
| Left (x = 0) | Tracer | Dirichlet | 1.0 mol/L |
| Right (x = nx) | Tracer | Neumann | 0 (zero-flux) |
| Top wall (z = nz) | Flow | Bounce-back (no-slip) | Solid wall |
| Bottom wall (z = 0) | Flow | Bounce-back (no-slip) | Solid wall |

## Validation Criteria

The reviewer should check the following:

1. **Transient: tracer front advection.** Early in the simulation, the tracer front should move from left to right at a rate consistent with the mean flow velocity. The front shape should show a parabolic leading edge (faster at the channel center, slower near the walls) due to the Poiseuille velocity profile.

2. **Transient: dispersive spreading.** The tracer front should spread as it advances, reflecting Taylor-Aris dispersion (a combination of molecular diffusion and velocity shear across the channel gap).

3. **Steady state: uniform concentration.** After sufficient time, the entire domain should reach a uniform tracer concentration of C = 1.0 mol/L. Since the tracer is non-reactive and continuously supplied at C = 1.0 at the inlet with a zero-flux outlet, the final state must be C = 1.0 everywhere.

4. **Flow field.** The velocity profile should be the same parabolic Poiseuille profile as in Scenario 01 (this can be verified independently).

5. **No negative concentrations.** All tracer values must remain non-negative throughout the simulation.

6. **Mass conservation.** The total tracer mass in the domain should increase monotonically until steady state is reached (no spurious sinks or sources).

## How to Use

### Using the CompLaB3D GUI

1. **Launch the GUI:**
   ```bash
   cd GUI
   python main.py
   ```

2. **Create a new project:**
   - Click **File > New Project**.
   - Select the **"Transport (Flow + Diffusion)"** template from the template list.
   - Choose a project name and output directory.

3. **Generate geometry:**
   - Go to the **Domain** panel.
   - Set dimensions to **nx=40, ny=15, nz=15** and **dx=1 um**.
   - Click **Generate Geometry** and select **Parallel Plates** as the medium type.
   - The geometry file will be saved to the `input/` folder.

4. **Verify settings:**
   - In the **Fluid** panel, confirm `delta_P = 2.0e-3` and `tau = 0.8`.
   - In the **Chemistry** panel, confirm there is 1 substrate named "Tracer" with D = 1.0e-9 m^2/s, left BC = Dirichlet (1.0), right BC = Neumann (0).
   - In the **General** panel, confirm `biotic_mode = false` and all kinetics toggles are off.

5. **Export and compile:**
   - Click **File > Export Project** to write `CompLaB.xml`.
   - Open a terminal, navigate to the `src/` directory, and run:
     ```bash
     make clean && make
     ```

6. **Run the simulation:**
   - Return to the GUI and go to the **Run** panel.
   - Click **Run Simulation**, or run from the terminal:
     ```bash
     ./complab3d ../path/to/your/CompLaB.xml
     ```

7. **Inspect results:**
   - Open the output VTK files in **ParaView**.
   - **Transient check:** Load an early time step and plot the tracer along the x-axis. Verify the advancing front with parabolic shape.
   - **Steady-state check:** Load the final time step. Confirm C = 1.0 uniformly across the domain.
   - Optionally, animate the VTK time series to visualize the tracer front moving through the channel.
