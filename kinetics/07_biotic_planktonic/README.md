# Scenario 7: Single-Species Planktonic Bacteria (LBM Solver)

## Overview

This scenario simulates a single microbial species suspended freely in the pore fluid as planktonic bacteria. Unlike the sessile biofilm in Scenario 6, planktonic bacteria are transported by flow and diffusion using an LBM advection-diffusion lattice. The bacteria consume dissolved organic carbon (DOC) via Monod kinetics and grow throughout the pore space.

## Reactions

**Monod growth kinetics:**

```
mu = mu_max * [DOC] / (Ks + [DOC])

dB/dt  = (mu - k_decay) * B        (biomass net growth)
dDOC/dt = -mu * B / Y              (substrate consumption)
```

Where:
- `mu` is the specific growth rate [1/s]
- `mu_max` is the maximum specific growth rate [1/s]
- `Ks` is the half-saturation constant [mol/L]
- `Y` is the yield coefficient [kg biomass / mol DOC]
- `k_decay` is the first-order decay rate [1/s]
- `B` is planktonic biomass density [kg/m3]

## Input Parameters

| Parameter | Value | Units | Description |
|-----------|-------|-------|-------------|
| `mu_max` | 0.05 | 1/s | Maximum specific growth rate |
| `Ks` | 1e-5 | mol/L | Half-saturation constant for DOC |
| `Y` | 0.4 | kg/mol | Yield coefficient |
| `k_decay` | 1e-7 | 1/s | First-order biomass decay rate |
| `C0_DOC` | 0.1 | mol/L | Initial DOC concentration |
| `D_pore` | 1e-9 | m2/s | DOC diffusion coefficient in pore space |
| `delta_P` | 2e-3 | (lattice) | Pressure gradient driving flow |
| `tau` | 0.8 | (dimensionless) | LBM relaxation parameter |
| `dx` | 1e-6 | m | Voxel size (1 um) |
| Initial biomass density | 10.0 | kg/m3 | Uniform starting density in pore space |

## Domain

- **Dimensions:** 30 x 15 x 15 voxels (NX x NY x NZ)
- **Geometry type:** Parallel plate channel (simple open pore space, no biofilm)
- **Voxel size:** dx = 1 um
- **Microbe location:** Uniformly distributed throughout the pore space (transported by flow)
- **Material numbers:**
  - 0 = Solid wall
  - 1 = Bounce-back interface (solid-fluid boundary)
  - 2 = Pore space (open fluid, where planktonic bacteria reside)

## Boundary Conditions

| Species | Left (Inlet) | Right (Outlet) | Walls |
|---------|--------------|-----------------|-------|
| DOC | Dirichlet: C = 0.1 mol/L | Neumann: dC/dx = 0 | No flux |
| Planktonic biomass | Dirichlet: B = 10.0 kg/m3 | Neumann: dB/dx = 0 | No flux |

**Flow:** Pressure-driven (delta_P = 2e-3) from left to right.

## Validation Criteria

A reviewer should check the following:

1. **Planktonic growth in pore space** -- Biomass density should increase over time in regions where DOC is available.
2. **DOC gradient along flow** -- DOC concentration should decrease along the flow direction (left to right) as bacteria consume it.
3. **Biomass transport** -- Planktonic bacteria should be advected with the flow and spread by diffusion; the biomass field should not be static.
4. **No negative concentrations** -- Both DOC and biomass must remain >= 0 everywhere.
5. **No biofilm on walls** -- Since there is no sessile component, biomass should exist only in pore space (material 2) and not attach to surfaces.
6. **Inlet replenishment** -- The Dirichlet boundary at the inlet should maintain a constant supply of DOC and planktonic biomass.

## How to Use

### Using the GUI (CompLaB Studio)

1. **Create project:** Open CompLaB Studio. Go to **File > New Project** and select template **7 - Planktonic Bacteria**.
2. **Review parameters:** Verify the Domain, Fluid, Chemistry, and Microbiology panels match the values listed above. Confirm the microbe solver is set to **LBM** (not CA) and the microbe type is **planktonic**.
3. **Generate geometry:** Go to **Tools > Geometry Generator**. Select **ABIOTIC Domain Generator** (option 1) since no biofilm patches are needed. Choose **Parallel Plates** as the medium type. Set dimensions to 30 x 15 x 15.
4. **Validate:** Click **Validate** in the Run panel. Fix any red errors before proceeding. Yellow warnings are informational.
5. **Export and compile:** Export the project. This generates the XML and .hh kinetics files. Recompile the solver:
   ```bash
   cd build && cmake .. && make -j$(nproc)
   ```
6. **Run:** Click **Run** in the GUI, or from the terminal:
   ```bash
   ./complab
   ```
7. **Inspect output:** Open the VTK files in the `output/` folder with ParaView. Check DOC concentration fields and planktonic biomass density fields to verify the validation criteria above.

### Using the Command Line

1. Copy the kinetics files from this folder to the source root:
   ```bash
   cp kinetics/07_biotic_planktonic/defineKinetics.hh src/
   cp kinetics/07_biotic_planktonic/defineAbioticKinetics.hh src/
   ```
2. Recompile:
   ```bash
   cd build && cmake .. && make -j$(nproc)
   ```
3. Generate geometry using the geometry generator:
   ```bash
   cd tools
   python geometry_generator.py
   # Select option 1 (ABIOTIC Domain Generator)
   # Parallel Plates, 30x15x15
   ```
4. Run the simulation:
   ```bash
   ./complab ../kinetics/07_biotic_planktonic/CompLaB.xml
   ```
5. Visualize results in ParaView.
