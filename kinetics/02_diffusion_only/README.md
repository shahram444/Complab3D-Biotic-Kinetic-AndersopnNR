# Scenario 02: Pure Diffusion (No Flow, No Reactions)

## Overview

This scenario tests **pure molecular diffusion** (Peclet number = 0) of a single tracer through an open pore domain. There is no pressure-driven flow and no chemical reactions. It validates that the LBM advection-diffusion solver correctly reproduces the analytical steady-state solution for one-dimensional diffusion with fixed concentration on one boundary and zero-flux on the other.

## Reactions

None. This is a passive diffusion test with no kinetics of any kind.

## Input Parameters

| Parameter | Value | Units | Description |
|-----------|-------|-------|-------------|
| `delta_P` | 0.0 | (lattice) | No pressure gradient (no flow) |
| `tau` | 0.8 | (lattice) | LBM relaxation time |
| `dx` | 1 | um | Voxel size |
| `D` (Tracer) | 1.0e-9 | m^2/s | Molecular diffusion coefficient |
| `biotic_mode` | false | -- | No microbes |
| `enable_kinetics` | false | -- | No biotic kinetics |
| `enable_abiotic_kinetics` | false | -- | No abiotic kinetics |
| `number_of_substrates` | 1 | -- | One passive tracer |

## Domain

| Property | Value |
|----------|-------|
| Dimensions (nx x ny x nz) | 30 x 10 x 10 |
| Geometry type | Open pore space (no internal obstacles) |
| Description | Entirely open domain filled with pore space (material number 2). No solid grains or walls obstructing diffusion. |
| Material numbers | 2 = pore space throughout |

## Boundary Conditions

| Boundary | Species | Type | Value |
|----------|---------|------|-------|
| Left (x = 0) | Tracer | Dirichlet | 1.0 mol/L |
| Right (x = nx) | Tracer | Neumann | 0 (zero-flux) |
| All other faces | Tracer | Periodic / no-flux | -- |

## Validation Criteria

The reviewer should check the following:

1. **Linear steady-state concentration profile.** At steady state, the tracer concentration should decrease linearly from C = 1.0 at the left boundary to C = 0.0 at the right boundary:
   ```
   C(x) = 1.0 * (1 - x/L)
   ```
   where L is the domain length (30 voxels).

2. **Left boundary value.** C = 1.0 at x = 0 (Dirichlet condition enforced).

3. **Midpoint value.** C approximately equals 0.5 at x = L/2.

4. **Right boundary value.** C approaches 0.0 at x = L.

5. **No flow field.** Velocity should be zero everywhere since `delta_P = 0`. The tracer should spread by diffusion only, not advection.

6. **Uniform cross-sections.** At any x-position, the concentration should be uniform across the y-z plane (no lateral gradients in an open domain).

7. **No negative concentrations.** All concentration values must remain non-negative.

## Expected Runtime

Approximately **1 minute** on a standard workstation (single core).

## How to Use

### Using the CompLaB3D GUI

1. **Launch the GUI:**
   ```bash
   cd GUI
   python main.py
   ```

2. **Create a new project:**
   - Click **File > New Project**.
   - Select the **"Diffusion Only"** template from the template list.
   - Choose a project name and output directory.

3. **Generate geometry:**
   - Go to the **Domain** panel.
   - Set dimensions to **nx=30, ny=10, nz=10** and **dx=1 um**.
   - Click **Generate Geometry** and select **Rectangular Channel** (option 1) to create a fully open pore domain.
   - The geometry file will be saved to the `input/` folder.

4. **Verify settings:**
   - In the **Fluid** panel, confirm `delta_P = 0.0` (this is critical -- any nonzero value will introduce flow).
   - In the **Chemistry** panel, confirm there is 1 substrate named "Tracer" with D = 1.0e-9 m^2/s, left BC = Dirichlet (1.0), right BC = Neumann (0).
   - In the **General** panel, confirm all kinetics toggles are off.

5. **Export and compile:**
   - Click **File > Export Project** to write `CompLaB.xml`.
   - Open a terminal, navigate to the `src/` directory, and run:
     ```bash
     make clean && make
     ```

6. **Run the simulation:**
   - Return to the GUI and go to the **Run** panel.
   - Click **Run Simulation**, or run from the terminal:
     ```bash
     ./complab3d ../path/to/your/CompLaB.xml
     ```

7. **Inspect results:**
   - Open the output VTK files in **ParaView**.
   - Plot the Tracer concentration along the x-axis at the domain center (y = 5, z = 5).
   - Confirm the linear profile from C = 1.0 (left) to C = 0.0 (right).
