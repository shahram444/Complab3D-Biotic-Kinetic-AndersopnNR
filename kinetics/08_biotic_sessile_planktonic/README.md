# Scenario 8: Sessile Biofilm (CA) + Planktonic Bacteria (LBM)

## Overview

This scenario combines both modes of microbial life in a single simulation: a sessile biofilm attached to the channel walls (solved by the Cellular Automaton) and planktonic bacteria freely suspended in the pore fluid (solved by an LBM advection-diffusion lattice). Both populations compete for the same dissolved organic carbon (DOC) substrate via Monod kinetics, demonstrating CompLaB3D's ability to couple CA and LBM microbial solvers simultaneously.

## Reactions

**Monod growth kinetics (both species consume DOC):**

```
Sessile:
  mu_s = mu_max_s * [DOC] / (Ks_s + [DOC])
  dB_s/dt  = (mu_s - k_decay_s) * B_s
  dDOC/dt -= mu_s * B_s / Y_s

Planktonic:
  mu_p = mu_max_p * [DOC] / (Ks_p + [DOC])
  dB_p/dt  = (mu_p - k_decay_p) * B_p
  dDOC/dt -= mu_p * B_p / Y_p

Total DOC consumption:
  dDOC/dt = -(mu_s * B_s / Y_s) - (mu_p * B_p / Y_p)
```

Where subscript `s` = sessile, `p` = planktonic.

## Domain

- **Dimensions:** 30 x 15 x 15 voxels (NX x NY x NZ)
- **Geometry type:** Parallel plate channel with biofilm patches on wall surfaces
- **Voxel size:** dx = 1 um
- **Sessile biofilm:** Attached to solid walls as patches (material 3 = core, 6 = fringe)
- **Planktonic bacteria:** Uniformly distributed in pore space (material 2)
- **Material numbers:**
  - 0 = Solid wall
  - 1 = Bounce-back interface
  - 2 = Pore space (planktonic bacteria reside here)
  - 3 = Microbe-1 Core (sessile biofilm interior)
  - 6 = Microbe-1 Fringe (sessile biofilm growth zone)

## Boundary Conditions

| Species | Left (Inlet) | Right (Outlet) | Walls |
|---------|--------------|-----------------|-------|
| DOC | Dirichlet: C = 0.1 mol/L | Neumann: dC/dx = 0 | No flux |
| Sessile biomass | N/A (CA solver) | N/A | Grows on walls |
| Planktonic biomass | Dirichlet: B = 5.0 kg/m3 | Neumann: dB/dx = 0 | No flux |

**Flow:** Pressure-driven (delta_P = 2e-3) from left to right.

## Validation Criteria

A reviewer should check the following:

1. **Both populations active** -- Both sessile and planktonic biomass fields should show growth over time. Neither population should collapse to zero (given sufficient DOC).
2. **DOC consumed by both** -- DOC concentration should decrease both near the biofilm walls (sessile consumption) and in the bulk fluid (planktonic consumption).
3. **Sessile dominance near walls** -- Near the biofilm surface, DOC depletion should be stronger due to the high sessile biomass density (99.0 vs. 5.0 kg/m3 initial).
4. **Planktonic transport** -- Planktonic biomass should be advected with the flow and diffuse; it should not be static.
5. **Competition effect** -- Total DOC consumption should be greater than in either single-species scenario (Scenario 6 or 7 alone) because both populations are consuming simultaneously.
6. **No negative concentrations** -- DOC and all biomass fields must remain >= 0 everywhere.
7. **Biofilm spreading** -- The CA solver should spread sessile biofilm into adjacent pore voxels when local biomass exceeds the division threshold.

## How to Use

### Using the GUI (CompLaB Studio)

1. **Create project:** Open CompLaB Studio. Go to **File > New Project** and select template **8 - Sessile + Planktonic**.
2. **Review parameters:** Verify the Domain, Fluid, and Chemistry panels. In the **Microbiology** panel, confirm there are two microbe species: one set to **CA** (sessile) and one set to **LBM** (planktonic), with the Monod parameters listed above.
3. **Generate geometry:** Go to **Tools > Geometry Generator**. Select **SESSILE BIOFILM Generator** (option 2). Choose **Parallel Plates** as the medium type. Set dimensions to 30 x 15 x 15. Choose a single-species scenario (e.g., scenario 8 for random patches), set biofilm thickness to 2, and coverage to 0.5.
4. **Validate:** Click **Validate** in the Run panel. Fix any red errors before proceeding.
5. **Export and compile:** Export the project. Recompile the solver:
   ```bash
   cd build && cmake .. && make -j$(nproc)
   ```
6. **Run:** Click **Run** in the GUI, or from the terminal:
   ```bash
   ./complab
   ```
7. **Inspect output:** Open the VTK files in the `output/` folder with ParaView. Check DOC, sessile biomass, and planktonic biomass fields. Compare DOC depletion patterns near walls vs. bulk fluid.

### Using the Command Line

1. Copy the kinetics files from this folder to the source root:
   ```bash
   cp kinetics/08_biotic_sessile_planktonic/defineKinetics.hh src/
   cp kinetics/08_biotic_sessile_planktonic/defineAbioticKinetics.hh src/
   ```
2. Recompile:
   ```bash
   cd build && cmake .. && make -j$(nproc)
   ```
3. Generate geometry using the geometry generator:
   ```bash
   cd tools
   python geometry_generator.py
   # Select option 2 (SESSILE BIOFILM Generator)
   # Parallel Plates, 30x15x15, 1 species, thickness=2, coverage=0.5
   ```
4. Run the simulation:
   ```bash
   ./complab ../kinetics/08_biotic_sessile_planktonic/CompLaB.xml
   ```
5. Visualize results in ParaView.
