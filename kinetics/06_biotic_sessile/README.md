# Scenario 6: Single-Species Sessile Biofilm (CA Solver)

## Overview

This scenario simulates a single microbial species attached to solid surfaces as a sessile biofilm. The biofilm grows and consumes dissolved organic carbon (DOC) following Monod kinetics. Biomass dynamics are handled by the Cellular Automaton (CA) solver, which spreads biofilm into adjacent pore voxels when local biomass exceeds a division threshold.

## Reactions

**Monod growth kinetics:**

```
mu = mu_max * [DOC] / (Ks + [DOC])

dB/dt  = (mu - k_decay) * B        (biomass net growth)
dDOC/dt = -mu * B / Y              (substrate consumption)
```

Where:
- `mu` is the specific growth rate [1/s]
- `mu_max` is the maximum specific growth rate [1/s]
- `Ks` is the half-saturation constant [mol/L]
- `Y` is the yield coefficient [kg biomass / mol DOC]
- `k_decay` is the first-order decay rate [1/s]
- `B` is biomass density [kg/m3]

## Input Parameters

| Parameter | Value | Units | Description |
|-----------|-------|-------|-------------|
| `mu_max` | 0.05 | 1/s | Maximum specific growth rate |
| `Ks` | 1e-5 | mol/L | Half-saturation constant for DOC |
| `Y` | 0.4 | kg/mol | Yield coefficient |
| `k_decay` | 1e-7 | 1/s | First-order biomass decay rate |
| `C0_DOC` | 0.1 | mol/L | Initial DOC concentration |
| `D_pore` | 1e-9 | m2/s | DOC diffusion coefficient in pore space |
| `D_biofilm` | 2e-10 | m2/s | DOC diffusion coefficient in biofilm |
| `delta_P` | 2e-3 | (lattice) | Pressure gradient driving flow |
| `tau` | 0.8 | (dimensionless) | LBM relaxation parameter |
| `dx` | 1e-6 | m | Voxel size (1 um) |
| Initial biomass density | 99.0 | kg/m3 | Starting density in biofilm voxels |

## Domain

- **Dimensions:** 30 x 15 x 15 voxels (NX x NY x NZ)
- **Geometry type:** Parallel plate channel with biofilm patches on wall surfaces
- **Voxel size:** dx = 1 um
- **Biofilm placement:** 1-2 patches of sessile biofilm on the solid walls
- **Material numbers:**
  - 0 = Solid wall
  - 1 = Bounce-back interface (solid-fluid boundary)
  - 2 = Pore space (open fluid)
  - 3 = Microbe-1 Core (dense interior biofilm)
  - 6 = Microbe-1 Fringe (active growth zone adjacent to pore)

## Boundary Conditions

| Species | Left (Inlet) | Right (Outlet) | Walls |
|---------|--------------|-----------------|-------|
| DOC | Dirichlet: C = 0.1 mol/L | Neumann: dC/dx = 0 | No flux |
| Biomass | N/A (CA solver, not transported) | N/A | Grows on walls |

**Flow:** Pressure-driven (delta_P = 2e-3) from left to right.

## Validation Criteria

A reviewer should check the following:

1. **DOC depletion near biofilm** -- DOC concentration should decrease in regions adjacent to biofilm patches compared to bulk fluid.
2. **Biomass growth** -- Biomass density should increase over time in biofilm voxels where DOC is available.
3. **Biofilm spreading** -- The CA solver should spread biofilm into neighboring pore voxels once local biomass exceeds the division threshold.
4. **No negative concentrations** -- DOC must remain >= 0 everywhere.
5. **Reduced diffusion in biofilm** -- DOC transport through biofilm should be slower than through open pore space (D_biofilm < D_pore).
6. **DOC gradient** -- A concentration gradient should develop from the pore center toward the biofilm surface.

## How to Use

### Using the GUI (CompLaB Studio)

1. **Create project:** Open CompLaB Studio. Go to **File > New Project** and select template **6 - Biofilm Sessile**.
2. **Review parameters:** Verify the Domain, Fluid, Chemistry, and Microbiology panels match the values listed above. The template comes pre-configured with sensible defaults.
3. **Generate geometry:** Go to **Tools > Geometry Generator**. Select **SESSILE BIOFILM Generator** (option 2). Choose **Parallel Plates** as the medium type. Set dimensions to 30 x 15 x 15. Choose a single-species scenario (e.g., scenario 8 for random patches), set biofilm thickness to 2, and coverage to 0.5.
4. **Validate:** Click **Validate** in the Run panel. Fix any red errors before proceeding. Yellow warnings are informational.
5. **Export and compile:** Export the project. This generates the XML and .hh kinetics files. Recompile the solver:
   ```bash
   cd build && cmake .. && make -j$(nproc)
   ```
6. **Run:** Click **Run** in the GUI, or from the terminal:
   ```bash
   ./complab
   ```
7. **Inspect output:** Open the VTK files in the `output/` folder with ParaView. Check DOC concentration fields and biomass density fields to verify the validation criteria above.

### Using the Command Line

1. Copy the kinetics files from this folder to the source root:
   ```bash
   cp kinetics/06_biotic_sessile/defineKinetics.hh src/
   cp kinetics/06_biotic_sessile/defineAbioticKinetics.hh src/
   ```
2. Recompile:
   ```bash
   cd build && cmake .. && make -j$(nproc)
   ```
3. Generate geometry using the geometry generator:
   ```bash
   cd tools
   python geometry_generator.py
   # Select option 2 (SESSILE BIOFILM Generator)
   # Parallel Plates, 30x15x15, 1 species, thickness=2, coverage=0.5
   ```
4. Run the simulation:
   ```bash
   ./complab ../kinetics/06_biotic_sessile/CompLaB.xml
   ```
5. Visualize results in ParaView.
