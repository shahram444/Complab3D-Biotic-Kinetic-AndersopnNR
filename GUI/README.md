# CompLaB Studio

Cross-platform GUI for CompLaB3D pore-scale reactive transport simulations.

## Installation

```bash
pip install -e ".[dev]"
complab-studio
```

## Requirements

- Python 3.10+
- PySide6
- numpy, vtk, matplotlib

## Testing

```bash
python -m pytest tests/ -v
```

See the main [README](../README.md) and [TESTING.md](../TESTING.md) for full details.
